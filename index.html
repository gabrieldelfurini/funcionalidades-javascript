Contador

O projeto é um simples contador implementado em HTML, CSS e JavaScript. Um número centralizado na tela é ajustado por três botões abaixo: um para decrementar o número, outro para zerá-lo e um terceiro para incrementá-lo. Essa interface interativa permite manipular o valor exibido de forma intuitiva.

Reviews

O projeto apresenta um painel de avaliações com a exibição central de uma pessoa e sua respectiva review. Três botões abaixo permitem navegar entre as avaliações, avançando ou retrocedendo, e um terceiro botão exibe aleatoriamente uma review na tela, proporcionando uma experiência dinâmica.

SideBar

O projeto é uma "aside bar" interativa com um botão de controle. Ao clicar no botão, a barra lateral é exibida ou ocultada, proporcionando uma experiência de navegação mais flexível e maximizando o espaço da tela quando não estiver em uso.

Modal

O projeto é um modal que surge ao clicar em um botão, proporcionando uma sobreposição focalizada de conteúdo. Ao acionar o botão, o modal é exibido, oferecendo informações adicionais, e desaparece quando o usuário opta por fechá-lo, garantindo uma interação amigável e não intrusiva.
